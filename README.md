# code.daysnap.cn

学习、复习、知识储备路线

## 项目操作

- 安装 `vitepress`

```sh
npm install -D vitepress
```

- 设置向导

```sh
npx vitepress init
```

- 开发运行

```sh
npm run docs:dev
```

## 集成插件

- vitepress-project-folder

[文档地址](https://vitepress-sandbox.js-bridge.com/)

```sh
npm i vitepress-plugin-sandpack -D
```
